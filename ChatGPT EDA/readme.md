# 🤖✨ ChatGPT Prompt & Response Analysis  
### *Exploring AI Conversations with Data & NLP*

---

## 🌟 Overview  
Ever wondered how ChatGPT responds to different types of prompts?  
This project dives deep into a large-scale dataset to uncover patterns in **AI-generated responses**, using **EDA + NLP techniques**.

🔍 From creative prompts to logical problem-solving — we analyze it all!

---

## 🎯 What This Project Does  
✅ Classifies prompt types (creative, listing, reasoning, etc.)  
✅ Analyzes response complexity using readability scores  
✅ Visualizes trends in AI-generated text  
✅ Extracts insights from 50K+ prompt-response pairs  

---

## 📊 Dataset Snapshot  
📁 **Dataset**: Alpaca GPT-4  
📦 **Records**: ~52,000  
🧾 **Fields**:
- `instruction`
- `input`
- `output`
- `text`

---

## 🧠 Key Concepts Used  
- 📌 Exploratory Data Analysis (EDA)  
- 📌 Text Processing & NLP  
- 📌 Readability Metrics (Flesch Score)  
- 📌 Data Visualization
